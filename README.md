
#  Personal Portfolio Website

A modern, fully responsive personal portfolio website built with **React.js** and **Tailwind CSS**, designed to showcase my skills, projects, and contact details in a professional manner.

**Features**
    Clean and modern UI
    Fully responsive for mobile, tablet, and desktop
    Colored header and Footer
    Footer contains social account links
    Interactive project showcase section
    Separate pages for Home, About, Projects,Skills and Contact
    Fast performance with React

 **Tech Stack**
    **React.js** – Frontend framework
    **Tailwind CSS** – Styling and responsiveness
    **React Router** – Navigation between pages
    **Netlify** – Deployment

**Folder Structure**
    MY PORTFOLIO/
    │── public/
    │── src/  **Main source code**
    │ │── assets/    **Images & icons**
    │ │── components/   **Reusable UI components ( Navbar, Footer, Buttons)**
    │ │── pages/   **Pages (Home, About, Projects,Skills, Contact)**
    │ │── App.jsx  **Main app component**
    │ │── main.jsx **Entry point; ; renders App.jsx to the DOM**
    │── index.html **Main HTML file for the app template**
    │── package.json  **Project metadata and dependencies**
    │── package-lock.json  **Exact versions of installed dependencies**
    │── tailwind.config.js  **Tailwind CSS configuration settings**
    │── postcss.config.js  **PostCSS configuration for processing CSS**
    │── vite.config.js  **Vite build tool configuration**
    │── README.md

**Install dependencies**
    npm install

**Run the development server**
    npm run dev
    
**Build for production**
    npm run build
